# AI Resume Screening System

An AI-powered Resume Screening and Applicant Tracking System (ATS) built using FastAPI, NLP, and Machine Learning concepts.

## Features

- Multi Resume Upload
- ATS Match Score Calculation
- Skill Extraction
- Missing Skill Detection
- Resume Keyword Extraction
- AI Resume Feedback
- AI Hiring Recommendation
- AI Interview Question Generation
- Resume Parsing
- PDF Report Generation
- Recruiter Dashboard UI

## Technologies Used

- Python
- FastAPI
- HTML
- CSS
- Jinja2
- Scikit-learn
- NLP
- TF-IDF
- Cosine Similarity
- PyMuPDF
- ReportLab

## AI/NLP Concepts Used

- TF-IDF Vectorization
- Cosine Similarity
- Keyword Extraction
- Rule-Based NLP
- Resume Parsing
